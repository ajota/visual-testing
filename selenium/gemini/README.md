# Configuración Gemini

Se debe tener en cuenta que para tener instalado esta herramienta es necesario tener instalado phyton ya que el modulo `node-gyp` el cual usa gemini necesita algunas caracteristicas de este para funcionar, teniendo eso en cuenta y si en algun momento encuentra problemas en la instalacion de gemini verifique lo siguiente:

Instale phyton preferiblemente siguiendo la siguiente guia: [node-gyp](https://github.com/nodejs/node-gyp#on-windows)

Ahí podra encontrar la informacion de instalacion correspondiente para el sistema operativo en el cual este trabajando.
